Prerequisites

  • Download Visual Boy Advance rr
  • Obtain a ROM of Pokemon FireRed US V1.1

Configuration

  • Do some stuff

Starting the game

  1. Start Visual Boy Advance rr
  2. Start the lua script
    1. Go to "Tools -> Lua Scripting -> New Lua Script Window
    2. Copy the lua script link and paste it in the Script File text box
    3. Click Run
  3. Start Pokemon FireRed
    1. Go to "File -> Open GBA Rom...
    2. Navigate to your Pokemon FireRed ROM
    3. Select the ROM
    4. Click "Open"
  4. Start the DoomRed bot with the "Start Bot" button under "Configuration"
